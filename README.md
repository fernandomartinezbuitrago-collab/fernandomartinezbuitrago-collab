### 2. [🧠 ASTRA OMNI: Framework Interactivo de Estrategia](https://poe.com/ASTRA_OMNI)
* **El Problema de Negocio:** Necesidad de una consultoría estratégica en tiempo real que se adapte dinámicamente a diferentes fases de un proyecto sin procesos rígidos.
* **La Solución Técnica:** Framework integrado en interfaz de chat diseñado con **prompts dinámicos** especializados. Implementa "Modos Dinámicos" que permiten al usuario interactuar con la IA de forma fluida para refinar ideas y estrategias.
* **Impacto:** Consultoría de alto nivel accesible al instante, permitiendo iteraciones rápidas sobre modelos de negocio complejos.

### 3. [🐶 Agentes de IA para Automatización de Soporte](https://github.com/fernandomartinezbuitrago-collab/enae_vet_es.git)
* **El Problema de Negocio:** Gestión manual de citas y soporte al cliente que saturaba la operativa diaria.
* **La Solución Técnica:** Desarrollo de agentes inteligentes integrados vía APIs para la gestión automatizada de calendarios y atención al cliente.
* **Impacto:** Reducción drástica de la carga administrativa y optimización de la agenda operativa.

### 4. [🎬 Sistema de Recomendación de Películas (Machine Learning)](https://github.com/fernandomartinezbuitrago-collab/Recomendador-de-Pel-culas.git)
* **El Problema de Negocio:** Parálisis por decisión del usuario y dificultad para filtrar contenido relevante en grandes bases de datos.
* **La Solución Técnica:** Motor híbrido de filtrado colaborativo (KNN) y basado en contenido utilizando Scikit-Learn y procesamiento de datos con Pandas.
* **Impacto:** Mejora en la personalización de la experiencia del usuario y optimización de sugerencias basadas en calidad y relevancia cinematográfica.
